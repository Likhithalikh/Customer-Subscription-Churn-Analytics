# Customer Subscription & Churn Analytics (SQL Project)

This project analyzes customer subscriptions and churn using SQL.  
It includes tables for customers, subscriptions, and churn records, along with sample data and analytics queries.

---

## Project Files

- `Customer-Subscription-Churn-Analytics.sql` : SQL script with tables, sample data, and queries  
- `Customer Subscription & Churn Analytics using SQL.png` : ER Diagram showing database structure  
- `README.md` : Project description  

---

## Database Tables

- **Customers** : customer information  
- **Subscriptions** : subscription details for each customer  
- **Churn** : records of customers who have left  

---

## Key Queries

- Calculate overall churn rate  
- Active vs churned customers  
- Churn reasons breakdown  
- Customer lifetime in days

- SQL
